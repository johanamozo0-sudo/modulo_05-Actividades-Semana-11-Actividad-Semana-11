El login es el proceso mediante el cual un usuario inicia sesión en una aplicación con sus credenciales. Esto permite verificar su identidad antes de acceder al sistema. Algunas rutas deben estar protegidas para que solo los usuarios autorizados puedan ver o modificar cierta información. De esta manera se evita el acceso no autorizado a funciones importantes. Además, no deben exponerse públicamente datos sensibles como contraseñas, documentos de identidad, direcciones, teléfonos o información personal de los usuarios.
